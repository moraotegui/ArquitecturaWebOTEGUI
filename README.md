# ArquitecturaWebOTEGUI
Gestor de turnos veterinarios
- `Clientes`
- `Turnos`
- `Veterinarios`
- `Mascotas`


## APIs:
- `Clientes y mascotas` gestion de los dueños y de las mascotas
- `Turnos` gestion de turnos
- `Veterinarios` gestionar la disponibilidad de los veterinarios

## Endpoints:

- `CLIENTES`
  - POST /clientes
  - GET /clientes
  - GET /clientes/:id
  - PUT /clientes/:id
  - DELETE /clientes/:id

- `TURNOS`
  - POST /turnos
  - GET /turnos
  - GET /turnos/:id
  - PUT /turnos/:id
  - DELETE /turnos/:id

- `VETERINARIOS`
  - POST /veterinarios
  - GET /veterinarios
  - GET /veterinarios/:id
  - PUT /veterinarios/:id
  - DELETE /veterinarios/:id
    
- `MASCOTAS`
  - POST /mascotas
  - GET /mascotas
  - GET /mascotas/:id
  - PUT /mascotas/:id
  - DELETE /mascotas/:id
